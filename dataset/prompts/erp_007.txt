Write an ABAP class with a static method that calculates the planned delivery date for a material. Working days are Monday to Friday, excluding weekends and the provided holidays in zholiday. If a material is not found in the list, the lead time defaults to 0.
Use the following Data Dictionary tables:
define table zmaterial {
    key mat_id : abap.char(20) not null;
    lead_time_days : abap.int4 not null;
}
define table zholiday {
  key holiday_date : abap.dats not null;
  name : abap.char(80);
}
Import:
order_date type string,
mat_id type string
Return:
delivery_date type date

Example:
Input:
order_date = 2024-12-22
mat_id = 'MAT1'
zmaterial = VALUE #( ( mat_id = 'MAT1' lead_time_days = 3 ) )
zholidays  = VALUE #( '2024-12-24'  '2024-12-25' )
Output:
delivery_date = '2024-12-27'
